# pico-vogel
A stop-the-pidgeon inspired game made in 560 characters with PICO-8 for TweetTweetJam3.

-------

STOP THE PIGEON!
A classic time-trial, catch the pigeon to get more time to catch more pigeons.

## Controls:
    Z: Accelerate
    X: Break
    ←→: Turn the plane
